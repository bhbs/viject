Viject âš¡ #

A tool for migrating your React app from react-scripts (Create React App) to Vite

Usage #

sh
cd <YOUR_APP>
npx viject

How it works #

  1. Rewrite npm scripts and dependencies for Vite
  2. Remove react-scripts from react-app-env.d.ts
  3. Rewrite react-app-env.d.ts for CRA compatibility
  4. Add vite.config.ts includes plugins for compatibility
  5. Add index.html
  6. Transform .js files including JSX into .jsx
